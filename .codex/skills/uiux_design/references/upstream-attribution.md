# Upstream Attribution And Adaptation

## Provenance

- Upstream repository: https://github.com/emilkowalski/skills
- Upstream file: `skills/apple-design/SKILL.md`
- Revision reviewed: `6bf24434f7730ad169077756cf9c7cd7bd675fc6`
- Reviewed on: 2026-07-20
- Upstream scope: Apple-inspired fluid interface principles translated for web technologies including CSS, Pointer Events, `requestAnimationFrame`, and Motion/Framer Motion.

This Starward Skill is an adaptation, not a verbatim installation. It retains the transferable principles of immediate response, direct manipulation, interruptibility, velocity continuity, bounded momentum, spatial consistency, restrained haptics, typography care, wayfinding, accessibility, and interactive prototyping. It maps them to React Native Gesture Handler, Reanimated, Expo/native haptics, React Native accessibility APIs, and Starward's product state.

It intentionally does not adopt upstream web code, fixed web constants, glass/translucent material direction, system-font default, or an iOS-identical Android experience. Starward's `DESIGN.md`, Source Plan, and Context control those conflicts.
