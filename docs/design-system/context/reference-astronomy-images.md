## 1. `其他.png`

### 产品与页面

- 产品：《今晚去观星》。
- 页面：①“今晚去观星”发现 / 附近观星点；②雁栖湖观星台详情；③天空条件；④装备与计划。

### 页面结构

- 发现页：状态栏 → 产品标题、地点和搜索 / 通知 → “附近观星点”地图卡 → “为你推荐”地点列表 → 固定五项底部导航。
- 详情页：沉浸式地点头图 → 地点名、推荐标记、地址和评分 → 距离 / 车程 → 导航路线 → 停车、厕所、人流量 → 安全提示 → 固定主按钮“导航去这里”。
- 天空条件页：标题和地点 / 更新时间 → 当前天气大卡 → 云量、透明度、视宁度三项卡 → 银河可见度、月相 → 日月升落信息。
- 装备页：标题 → 三项分段导航 → 3×3 装备清单 → 建议到达时间卡。

### 信息层级与字体

- 一级信息是页面标题、地点名、当前温度和主操作按钮；使用较粗、较大的无衬线字。
- 二级信息是推荐地点名、距离 / 车程、云量 / 透明度 / 视宁度、到达时间；字号中等，数字比标签更醒目。
- 三级信息是地址、安全提示、设备状态和辅助说明；使用较小字号与蓝灰色文字。
- 评分由星形符号、数值和推荐胶囊共同形成高辨识度状态。

### 导航方式

- 发现页使用固定五项底部导航：主页、观星点、天空、装备、我的；当前项用蓝色图标和文字高亮。
- 详情页使用左上返回、右上分享，并把“导航去这里”作为底部固定主操作。
- 装备页顶部用“装备清单 / 计划建议 / 观星目标”分段切换。

### 卡片结构

- 地图被放在大圆角独立卡中；推荐地点用图片缩略图 + 名称 + 距离 / 车程 + 评分的横向列表卡。
- 地点详情的白色内容面板覆盖在头图下方，内部用浅灰分组块组织路线、设施与安全信息。
- 天空条件将结论大卡、三项指标小卡和双列信息卡组合使用。
- 装备以等宽方格卡陈列，每格包含实物图、名称与优先级文字。

### 地图交互与底部浮层

- 地图卡可直接观察到多个地点标记、当前定位点和一个被放大的望远镜标记；标记大小体现选中状态。
- 本图未展示地图上的路线操作控件，也未展示可拖动 Bottom Sheet。
- 发现页地图与下方推荐列表分离；详情页底部主按钮固定，属于操作栏而不是地图浮层。

### 圆角、间距、边框与阴影

- 手机内页面以中到大圆角卡片为主，地图、详情面板和主按钮圆角一致性较强。
- 页面左右留白稳定，卡片间距紧凑但分组清楚；小卡之间使用窄间隙。
- 边框很弱，主要依靠白色 / 浅灰表面深浅区分层级。
- 阴影柔和，集中在浮于深色背景上的白色大面板和主按钮；无重阴影。

### 主要颜色与深浅模式

- 深靛蓝页面背景，白色和浅灰卡片，近黑文字，亮蓝作为选中与主操作色。
- 黄色只用于评分星，绿色用于设施状态和“适宜观星”状态。
- 属于深色外壳与浅色内容面板混合的规划模式，不是纯黑夜间模式。

### 可复用组件

- 地点上下文栏、搜索按钮、通知按钮。
- 观星点地图卡与状态化地图标记。
- 地点推荐列表卡、评分与推荐胶囊。
- 地点详情头图、路线摘要、设施三联卡、安全提示卡。
- 观星条件结论卡、三项评分卡、日月升落条。
- 装备分段导航、装备方格、到达时间建议卡。
- 固定五项底部导航与底部主操作。

## 2. `日间模式.png`

### 产品与页面

- 产品：《今晚去观星》。
- 页面：①观星地图；②星野湖畔观星点详情；③天空天气；④装备与计划。

### 页面结构

- 观星地图：顶栏 → 四项筛选胶囊 → 全屏地图、路线和地点标记 → 右侧图层 / 定位控件 → 底部地点预览卡。
- 详情页：顶栏 → 地点实景轮播 → 名称、推荐、评分和评价数 → 距离 / 驾车时间 / 暗空指数 → 设施服务 → 说明 → 固定“开始导航”。
- 天空天气：标题和地点 → 今晚观星条件评分仪表 → 云量 / 透明度 / 视宁度 → 银河可见度和月相 → 日月升落 → 逐小时预报。
- 装备与计划：标题 → 推荐到达时间 → 装备清单网格 → 出行清单。

### 信息层级与字体

- 地图和地点实景承担第一视觉层；地点名、9.2 评分、20:30 到达时间和蓝色主按钮为第一信息层。
- 距离、时间、暗空指数与设施图标构成第二层。
- 描述、设备说明、逐小时天气与清单状态是第三层。
- 无衬线字体；标题偏粗，核心数字显著放大，小标签与辅助文案使用冷蓝灰色。

### 导航方式

- 地图页和天空天气页使用左上返回；地图页右上显示“我的位置”。
- 地图筛选通过“暗空指数 / 驾车时间 / 设施 / 天气”四个胶囊入口完成。
- 详情页使用返回、收藏和分享；主操作固定在底部。
- 这组画面未显示全局底部导航，页面更偏任务流。

### 卡片结构

- 地图下方的地点预览卡近似底部浮层：左图右文，上部显示评分与标签，下部是距离 / 时间和全宽主按钮。
- 详情页采用实景大图加白色内容面板，设施用五个等宽图标小卡。
- 天气页是一张纵向连续信息面板，仪表盘、摘要卡和小时预报连续排列，避免过度嵌套。
- 装备页使用推荐时间大卡、2×3 装备图卡和清单面板。

### 地图交互与底部浮层

- 地图可见多个地点标记、当前路线、车辆位置、起终点与右侧图层 / 定位按钮。
- 选中的“星野湖畔”标记与下方同名地点预览卡保持一致，显示地图—卡片联动。
- 预览卡贴在地图底部，视觉上是低位 Bottom Sheet；图中无拖拽柄，无法确认是否支持拖动。

### 圆角、间距、边框与阴影

- 整体使用约中等圆角；地图预览卡、实景图、信息面板、设施小卡和按钮保持同一圆角语言。
- 白色背景留白充足，卡片间距规整；装备网格的间隙小于模块间距。
- 多数边框为极浅蓝灰细线；内部指标依靠竖向分隔线组织。
- 手机内阴影很轻，地点浮层和主按钮略有抬升；展示板外壳的蓝色发光不属于产品 UI。

### 主要颜色与深浅模式

- 纯白与极浅冷蓝为主背景，深蓝文字，亮蓝用于按钮、路线、图标和选中状态。
- 黄色用于评分和星级，绿色只出现在积极状态文字。
- 属于明亮日间规划模式。

### 可复用组件

- 地图筛选胶囊组、地图标记、路线、图层与定位按钮。
- 地图 Bottom Sheet / 地点预览卡。
- 地点头图轮播、评分摘要、设施服务图标组。
- 观星评分仪表、天气三指标、银河与月相双卡、日月升落条、逐小时预报。
- 推荐到达时间卡、装备网格、出行清单、固定导航主按钮。

## 跨图片可复用证据总结

### 页面结构

- 六张图共同覆盖一条稳定的用户路径：找地点 → 看详情与设施 → 看天气 / 天象条件 → 规划路线与到达时间 → 准备装备与清单。
- 地图、地点详情、天气 / 天象、装备 / 计划是重复频率最高的四类代表页面。
- 深色、浅色与红光模式大多保持相同信息架构，说明模式切换优先改变视觉与亮度，不改变核心任务顺序。

### 信息层级

- 第一层：是否适合观星、观星指数、选中地点、最佳时间和主操作。
- 第二层：距离、车程、暗空指数、设施、云量、透明度、视宁度、银河可见度与月相。
- 第三层：逐小时数据、评价分布、安全提示、装备说明和清单细项。
- 图像、地图、大数字、连续时间带和主按钮承担最高视觉权重。

### 导航方式

- 两种全局导航证据并存：固定五项底部导航，以及不显示全局导航的专注任务流。
- 页面级导航稳定采用左上返回、右上位置 / 收藏 / 分享。
- 筛选和视图切换普遍采用横向胶囊或分段控制。
- 从地图到详情再到导航，主要通过选中标记、底部地点预览卡和主按钮推进。

### 卡片与 Bottom Sheet

- 消费级结论和行动信息使用中等圆角大卡；设施、装备和关键指标使用等宽网格。
- 专业数据倾向连续数据区、时间轴或矩阵，不将每个数值拆成独立卡片。
- 地图页反复出现贴底的地点预览面板，可作为 Bottom Sheet 结构证据；所有截图都没有清晰拖拽柄，因此不能仅凭图片确认拖拽手势。

### 地图交互

- 地图证据包括：地点标记、选中态、评分气泡、当前位置、起终点、车辆、路线、定位、图层、筛选和路线偏好。
- 地图选中点与底部地点卡在名称和状态上保持一致；详情页继续使用同一地点实景和关键指标。
- 路线规划页展示了起终点输入、距离 / 车程 / 到达时间、路线地图、路线工具和导航按钮的完整闭环。

### 字体、圆角、间距、边框与阴影

- 字体均为无衬线；标题、地点名和关键数字使用粗体 / 大字号，单位、标签和说明显著缩小。
- 大卡和大图使用中等圆角，筛选器、标签和小按钮使用胶囊圆角。
- 页面边距稳定，卡内间距小于模块间距；数据网格通过列对齐和细分隔线控制密度。
- 边框以低对比冷灰 / 蓝灰 / 暗红细线为主；阴影克制，主要用于地图浮层、浮动按钮和大面板。

### 颜色与模式

- 日间规划：白色 / 冰蓝背景、深蓝文字、亮蓝主操作，黄色评分，绿色积极状态。
- 深色规划：深靛蓝 / 蓝黑背景、冷白文字、多级深蓝表面、亮蓝交互，黄 / 绿仅作状态信号。
- 红光现场：近黑背景与单一暖红前景，所有信息通过红色明度和字重分层，避免高亮白与蓝光。
- 地点实景、天空、地图和产品图始终是主要质感来源；UI 表面相对克制。
