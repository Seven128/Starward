## 跨图片可复用证据总结

### 界面结构

- 主要由“全屏地图或大面积留白背景 + 白色内容表面”组成。
- 详情页使用圆角内容层覆盖彩色头部；地图页使用悬浮筛选与 Bottom Sheet。
- 固定底部操作、吸顶分段导航和地图浮层在不同页面保持一致的胶囊语法。

### 信息层级

- 展示型页面先给行程名称或当前地点，再给日期 / 地点序列，最后给天气、里程、时间等辅助信息。
- 编辑型页面将“保存”提升为最高优先级，同时弱化未选状态和排序控件。
- 地图页通过“筛选状态 → DAY 标签 / 编号节点 → 地图底图文字”建立覆盖层级。

### 导航方式

- 系统级：返回或关闭。
- 页面级：总览 / DAY 1 / DAY 2 / 待计划分段导航。
- 内容级：横向行程卡、横向地点卡、DAY 摘要卡进入箭头。
- 地图级：全程 / 分日切换、编号节点、回中按钮、地图入口。

### 卡片与浮层模式

- 行程卡、DAY 卡、地点卡和输入卡均使用白色或浅灰实体表面、大圆角、极浅边框 / 柔影。
- Bottom Sheet 有两种密度：地点联动的短浮层、编辑行程的高浮层。
- 浮层始终完全位于屏幕内，并保留足够地图上下文；高浮层提供明确拖拽把手。

### 字体与视觉层级

- 无衬线黑体；大标题和关键地点名使用粗体，内容标题使用中粗体，辅助信息使用浅灰常规字重。
- 颜色使用克制，主要靠字号、字重、留白、实体表面和选中态黑白反转建立层级。

### 圆角、间距、边框和阴影

- 大容器视觉圆角集中在约 20–28px；按钮与筛选器使用全圆角；图片使用中等圆角。
- 页面模块采用宽松纵向留白，列表卡片之间更紧凑。
- 边框通常为极浅灰 1px 视觉重量；阴影柔和、扩散大、对比低。

### 主要颜色与深浅模式

- 6 张截图全部为明亮模式。
- 主色为白、近黑、冷浅灰；地图路线使用浅蓝 / 青蓝，行程详情头部使用低饱和淡绿色。
- 黑色仅用于主操作、当前筛选、当前节点和当前分页等高信号状态。
- 实景图片负责提供自然环境色，界面本身保持低彩度。

- 胶囊分段导航与地图筛选器。
- 横向行程卡、DAY 摘要卡、待计划卡。
- 图片空间卡、实景地点卡、天气摘要卡。

- 路线地图预览与全屏地图。
